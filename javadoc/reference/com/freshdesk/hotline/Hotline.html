Constants

public static final String HOTLINE_ACTION_UNREAD_MESSAGE_COUNT_CHANGED

Since: API Level 1.0.1

Action String used by the local broadcast sent by Hotline SDK for any change in unread count of messages. Note : This will be a local broadcast and has to be registered through LocalBroadcastManager Eg: IntentFilter intentFilter = new IntentFilter(Hotline.HOTLINE_ACTION_UNREAD_MESSAGE_COUNT_CHANGED); LocalBroadcastManager.getInstance(this).registerReceiver(receiver, intentFilter);

Constant Value: "HotlineUnreadMessageCountChanged"

public Hotline linkifyWithPattern (String regex, String defaultScheme)

Since: API Level 1.0.0

Creates deep links in messages turning the matches into links based on the regex. Internally uses addLinks(Spannable, Pattern, String) The given regex is converted into a pattern and passed to the addLinks method.

Parameters

defaultScheme Url scheme string (eg http:// to be prepended to the url of links that do not have a scheme specified in the link text)
